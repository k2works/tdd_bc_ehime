Quick Start

以下のリンクからクラウドIDEを起動する

IDEが起動したらターミナルに以下のコマンドを入力する

npm install

パッケージのインストールが終わったらsrcフォルダにあるindex.htmlファイルを開いて画面右上にあるOpen Preview to the Sideボタンを押してプレビュー画面を開く


<script>タグ内のJavaScriptコードを編集して開発を進める
